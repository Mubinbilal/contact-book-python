# contact-book-python
This is a simple command-line based Contact Book application built using Python. It allows users to store, manage, and retrieve contact information easily.  The application uses a JSON file to save data, so all contacts remain محفوظ (saved) even after closing the program.
🚀 Features
➕ Add new contacts (Name, Phone, Email)
📋 View all saved contacts
🔍 Search contacts by name
🗑️ Delete contacts
💾 Persistent storage using JSON file
🛠️ Technologies Used
Python
JSON (for data storage)
▶️ How to Run
Make sure Python is installed
Download or clone this repository
Run the file:
python contact_book.py
📂 File Structure
contact_book.py
contacts.json  (auto-created)
